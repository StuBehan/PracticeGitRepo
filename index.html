Git commands

  • add .
    • adds all changes to the staging area
  • commit -m "commit comment"
    • commits changes to the master or active branch
  • log --online
    • shows a log of commits keeping them to one line each
  • checkout /commit serial/
    • shows the master branch at that specfic commit - read only
  • checkout master (or /branch name/)
    • returns to the specified branch
  • revert /commit serial/
    • reverts the specified commit, removes the changes
  • reset /commit serial/
    • resets the repo to the specified commit, removes all other changes, not reverseable!
  • reset /commit serial/
    • resets the repo to the specified commit, removes all other changes, not reverseable!
  • branch /branch name/
    • creates a branch of the specified name
  • branch -a
    • shows a list of all branches
